# 世界地图 + 区域植被 + 海岸修复

## 三个需求

### 1. 真实世界地图板块
- 用 Canvas 绘制简化七大洲轮廓作为陆地蒙版
- 每个顶点算经纬度 → 查蒙版 → 判断陆地/海洋
- 高度用 simplex noise 叠加在陆地区域
- 保持当前低多边形卡通风格

### 2. 区域植被差异化
按纬度+大陆分区：
- **热带**（赤道±20°）：高大泪滴树+棕榈，深绿
- **温带**（20°-50°）：中等锥形树，翠绿+秋色橙
- **寒带/针叶林**（50°-70°）：细高针叶树，墨绿
- **极地**（>70°）：无树，雪地
- **沙漠**（撒哈拉、澳洲内陆）：极少植被，沙色地面
- 山脉（高海拔）：灰色岩石锥 + 雪顶

### 3. 海岸暗色带修复
- 去掉地形海洋顶点的缩进
- 海洋网格和地形共享同一半径
- 海洋 renderOrder 设低，陆地自然覆盖
- 夜间 brightness 下限提到 40%

## 修改文件
- `src/globe/terrain.ts` — 替换 noise 为世界地图蒙版
- `src/globe/Ocean.ts` — 调整半径
- `src/features/Trees.ts` — 按纬度分区不同树型
- `src/systems/DayNightCycle.ts` — 夜间亮度
